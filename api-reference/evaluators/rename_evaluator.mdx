---
title: 'Rename Evaluator'
api: 'POST /v0/evaluator/rename'
---
Renames an evaluator from `name` to `new_name` in your account.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Query Parameters

<ParamField query="name" type="string" required="true">
Name of the evaluator to rename.
</ParamField>

<ParamField query="new_name" type="string" required="true">
New name for the evaluator.
</ParamField>

<RequestExample>

```bash cURL
curl --request POST \
  --url 'https://api.unify.ai/v0/evaluator/rename?name=eval1&new_name=eval2' \
  --header "Authorization: Bearer $UNIFY_KEY"
```

```python Python
import requests

url = "https://api.unify.ai/v0/evaluator/rename?name=eval1&new_name=eval2"

headers = {"Authorization": "Bearer <token>"}

response = requests.request("POST", url, headers=headers)

print(response.text)
```
