---
title: 'Add Dataset Entries'
api: 'POST /v0/datasetv2/{name}/entries'
---
Add multiple entries to a dataset.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="name" type="string" required="true">
Dataset name (can include forward slashes)
</ParamField>

#### Body

<ParamField body="name" type="string" required="true">
Dataset name (can include forward slashes)
</ParamField>

<RequestExample>

```bash cURL
curl --request POST \
  --url 'https://api.unify.ai/v0/datasetv2/my_dataset/entries' \
  --header "Authorization: Bearer $UNIFY_KEY"
  --header 'Content-Type: application/json' \
  --data '{
    "name": "my_dataset"
}'
```

```python Python
import requests

url = "https://api.unify.ai/v0/datasetv2/my_dataset/entries"

headers = {"Authorization": "Bearer <token>"}

json_input = {"name": "my_dataset"}

response = requests.request("POST", url, json=json_input, headers=headers)

print(response.text)
```

</RequestExample>
