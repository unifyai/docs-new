---
title: 'Rename Custom Api Key'
api: 'POST /v0/custom_api_key/rename'
---
Renames a custom API key from a LLM provider in your account.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Query Parameters

<ParamField query="key" type="string" required="true">
Name of the custom API key to be updated.
</ParamField>

<ParamField query="new_key" type="string" required="true">
New name for the custom API key.
</ParamField>

<RequestExample>

```bash cURL
curl --request POST \
  --url 'https://api.unify.ai/v0/custom_api_key/rename?key=key1&new_key=key2' \
  --header 'Authorization: Bearer <UNIFY_KEY>'
```

```python Python
import requests

url = "https://api.unify.ai/v0/custom_api_key/rename?key=key1&new_key=key2"

headers = {"Authorization": "Bearer <token>"}

response = requests.request("POST", url, headers=headers)

print(response.text)
```

</RequestExample>
