---
title: 'Delete Router'
api: 'DELETE /v0/router'
---
Deletes a specific trained router, as well as all the training files etc.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Query Parameters

<ParamField query="name" type="string" required="true">
Name of the router to delete.
</ParamField>

<RequestExample>

```bash cURL
curl --request DELETE \
  --url 'https://api.unify.ai/v0/router?name=my_router' \
  --header "Authorization: Bearer $UNIFY_KEY"
```

```python Python
import requests

url = "https://api.unify.ai/v0/router?name=my_router"

headers = {"Authorization": "Bearer <token>"}

response = requests.request("DELETE", url, headers=headers)

print(response.text)
```

</RequestExample>
