title: 'Add Prompt'
api: 'PUT /v0/dataset/add_prompt'
---


#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Body

<ParamField body="name" type="string" required="true">
Name of the dataset to add to
</ParamField>

<ParamField body="prompt_data" type="object" required="true">
JSON object containing the prompt data to upload.
</ParamField>

<RequestExample>

```bash cURL
curl --request PUT \
  --url 'https://api.unify.ai/v0/dataset/add_prompt' \
  --header 'Authorization: Bearer <UNIFY_KEY>' \
  --header 'Content-Type: application/json' \
  --data '{}'
```

```python Python
import requests

url = "https://api.unify.ai/v0/dataset/add_prompt"

headers = {"Authorization": "Bearer <token>"}

json_input = {}

response = requests.request("PUT", url, json=json_input, headers=headers)

print(response.text)
```
