---
title: 'Add Prompt'
api: 'POST /v0/dataset/add_prompt'
---


#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Body

<ParamField body="name" type="string" required="true">
Name of the dataset to add to
</ParamField>

<ParamField body="prompt_data" type="object" required="true">
JSON object containing the prompt data to upload.
</ParamField>

<RequestExample>

```bash cURL
curl --request POST \
  --url 'https://api.unify.ai/v0/dataset/add_prompt' \
  --header "Authorization: Bearer $UNIFY_KEY" \
  --header 'Content-Type: application/json' \
  --data '{
    "name": "dataset_1",
    "prompt_data": {
        "prompt": {
            "messages": [
                {
                    "content": "What is the capital of Spain?",
                    "role": "user"
                }
            ]
        },
        "ref_answer": "Madrid"
    }
}'
```

```python Python
import requests

url = "https://api.unify.ai/v0/dataset/add_prompt"

headers = {"Authorization": "Bearer <token>"}

json_input = {"name": "dataset_1", "prompt_data": {"prompt": {"messages": [{"content": "What is the capital of Spain?", "role": "user"}]}, "ref_answer": "Madrid"}}

response = requests.request("POST", url, json=json_input, headers=headers)

print(response.text)
```
