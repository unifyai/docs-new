---
title: 'Get Log Metrics'
api: 'GET /v0/logs/metrics'
---
Returns the reduction metric for filtered values for a specific key from a project.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Query Parameters

<ParamField query="project" type="string" required="true">
Name of the project to get logs from.
</ParamField>

<ParamField query="key" type="string" required="true">
The key you would like to extract the reduction metric for.
</ParamField>

<ParamField query="metric" type="string" required="true">
The name of the metric you would like to compute.
</ParamField>

<ParamField query="filter_expr" type="string | null" >
Boolean string to filter logs. TODO: Detailed page.
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url 'https://api.unify.ai/v0/logs/metrics?project=eval-project&key=score&metric=mean&filter_expr=len(output)%20>%20200%20and%20temperature%20==%200.5' \
  --header "Authorization: Bearer $UNIFY_KEY"
```

```python Python
import requests

url = "https://api.unify.ai/v0/logs/metrics?project=eval-project&key=score&metric=mean&filter_expr=len(output)%20>%20200%20and%20temperature%20==%200.5"

headers = {"Authorization": "Bearer <token>"}

response = requests.request("GET", url, headers=headers)

print(response.text)
```
