---
title: 'Delete Default Prompt'
api: 'DELETE /v0/default_prompt'
---
Deletes a default prompt from your account.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Query Parameters

<ParamField query="name" type="string" required="true">
Name of the default_prompt to delete.
</ParamField>

<RequestExample>

```bash cURL
curl --request DELETE \
  --url 'https://api.unify.ai/v0/default_prompt?name=default_prompt_1' \
  --header 'Authorization: Bearer <UNIFY_KEY>'
```

```python Python
import requests

url = "https://api.unify.ai/v0/default_prompt?name=default_prompt_1"

headers = {"Authorization": "Bearer <token>"}

response = requests.request("DELETE", url, headers=headers)

print(response.text)
```

</RequestExample>
