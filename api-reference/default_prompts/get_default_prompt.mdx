---
title: 'Get Default Prompt'
api: 'GET /v0/default_prompt'
---
Returns the name and prompt of a default prompt from your account. The configuration
contains the same information as the arguments passed to the `POST` function of
`/v0/default_prompt`.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Query Parameters

<ParamField query="name" type="string" required="true">
Name of the default prompt to return the configuration of.
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url 'https://api.unify.ai/v0/default_prompt?name=default_prompt_1' \
  --header "Authorization: Bearer $UNIFY_KEY"
```

```python Python
import requests

url = "https://api.unify.ai/v0/default_prompt?name=default_prompt_1"

headers = {"Authorization": "Bearer <token>"}

response = requests.request("GET", url, headers=headers)

print(response.text)
```

</RequestExample>
