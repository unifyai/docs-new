### \_\_init\_\_

```python
def __init__(endpoint: Optional[str] = None,
             model: Optional[str] = None,
             provider: Optional[str] = None,
             api_key: Optional[str] = None) -> None
```

Initializes the ChatBot object.

**Arguments**:

- `endpoint` _str, optional_ - Endpoint name in OpenAI API format:
  \<uploaded_by\>/\<model_name\>@\<provider_name\>
  Defaults to None.
  
- `model` _str, optional_ - Name of the model. If None,
  endpoint must be provided.
  
- `provider` _str, optional_ - Name of the provider. If None,
  endpoint must be provided.
  
- `api_key` _str, optional_ - API key for accessing the Unify API.
  If None, it attempts to retrieve the API key from the
  environment variable UNIFY_KEY.
  Defaults to None.
  

**Raises**:

- `UnifyError` - If the API key is missing.
